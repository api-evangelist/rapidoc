# RapiDoc (rapidoc)

RapiDoc is a web component that allows developers to easily integrate interactive documentation for their APIs. It provides a user-friendly interface for exploring and testing API endpoints, displaying detailed information about request and response parameters, and offering code examples in multiple programming languages. RapiDoc also supports authentication methods, response validation, and custom theming options to tailor the documentation to a specific brand or project.

## APIs

### RapiDoc

RapiDoc is a fast, embeddable web component that renders OpenAPI (Swagger) 2.0/3.x specifications as interactive API documentation. Drop the `<rapi-doc>` element into any web page to get a searchable, navigable UI with endpoint details, request/response schemas, examples, and code snippets. It includes a built-in "Try It" console with auth support (API key, Bearer, Basic, OAuth), server selection, and environment variables, so developers can test calls directly from the docs.
